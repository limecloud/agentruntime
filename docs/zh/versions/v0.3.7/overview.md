# v0.3.7 概览

Agent Runtime v0.3.7 是标准生态发现补丁版本。它把 Agent Artifact 提升到当前 standards map，并刷新 README、版本导航和 LLM entrypoints。

## Included

- 当前标准表加入 Agent Artifact 链接。
- boundary map 增加 durable deliverable layer。
- 刷新 AI clients 使用的 LLM entrypoints。
- 不改变 Agent Runtime 核心协议语义。
