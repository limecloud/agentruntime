# Agent Runtime v0.1.0 规范

v0.1.0 的规范快照记录初始基线。当前 [latest 规范](/zh/specification) 已在此基线之上继续扩展。

## v0.1.0 范围

- Runtime identity 与 ownership boundaries。
- Typed runtime event stream。
- Runtime control plane。
- Tool、context、model、policy 与 subagent coordination facts。
- Durable snapshots 与 read models。
- Evidence 与 replay export boundaries。

完整 runtime 规则见 [latest 规范](/zh/specification)。
