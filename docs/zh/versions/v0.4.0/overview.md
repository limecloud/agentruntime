# Agent Runtime v0.4.0

Agent Runtime v0.4.0 引入公开标准加 product profile 的双轨模型。公开 Agent Runtime core 继续保持跨 runtime 可移植，Lime AgentRuntime Profile 则作为 Lime current runtime 的严格 conformance layer。

## 重点

- 新增英文与简体中文 Lime AgentRuntime Profile。
- 新增更严格的 Lime profile event 与 snapshot schemas。
- 新增 submit turn、tool approval、task retry、routing single candidate、evidence export 与 thread read snapshot fixtures。
- 明确 public core、recommended extensions 与 product profile conformance layers。
- 刷新 README、导航、release notes、package metadata 与 LLM entrypoints 到 v0.4.0。
