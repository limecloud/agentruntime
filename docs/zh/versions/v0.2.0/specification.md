# Agent Runtime v0.2.0 规范

v0.2.0 的规范快照与当前 [latest 规范](/zh/specification) 对齐。

## v0.2.0 范围

- Runtime identity 与 ownership boundaries。
- Typed runtime event stream 与扩展 event families。
- Runtime control plane 与 durable read models。
- Tool、context、model、policy、permission、sandbox、hook 和 process facts。
- Model routing、candidate sets、fallback、cost、quota 和 rate-limit facts。
- Subagent graph、durable jobs、remote channel recovery、telemetry correlation、large output refs 和 history reconstruction。
- Evidence、replay、review 与 signal applicability boundaries。

完整 runtime 规则见 [latest 规范](/zh/specification)。
