# v0.3.8 概览

Agent Runtime v0.3.8 是标准生态发现 patch release。它把 Agent Tool 提升为当前标准，并刷新 README、版本导航和 LLM 入口。

## 包含内容

- Current standards 表增加 Agent Tool。
- Boundary map 增加 portable tool layer。
- 面向 AI clients 的 LLM entrypoint refresh。
- 不改变 Agent Runtime 的核心协议语义。
