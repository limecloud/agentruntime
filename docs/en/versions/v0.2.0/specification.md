# Agent Runtime v0.2.0 Specification

The v0.2.0 specification snapshot is aligned with the current [latest specification](/en/specification).

## v0.2.0 scope

- Runtime identity and ownership boundaries.
- Typed runtime event stream and expanded event families.
- Runtime control plane and durable read models.
- Tool, context, model, policy, permission, sandbox, hook, and process facts.
- Model routing, candidate sets, fallback, cost, quota, and rate-limit facts.
- Subagent graph, durable jobs, remote channel recovery, telemetry correlation, large output refs, and history reconstruction.
- Evidence, replay, review, and signal applicability boundaries.

See the [latest specification](/en/specification) for the full runtime rules.
