# v0.3.4 specification snapshot

v0.3.4 is a documentation homepage correction release. The core Agent Runtime specification remains compatible with v0.3.3.

## Homepage contract

Compatible documentation SHOULD expose localized home pages as landing pages with:

- `layout: home` frontmatter for `/en/` and `/zh/` index routes.
- Clear specification, quickstart, ecosystem, and LLM-context actions.
- Concise standard ownership and start-here links.
- No sidebar-only overview page as the locale root.
