# Agent Runtime v0.4.0

Agent Runtime v0.4.0 introduces the public-standard plus product-profile model. It keeps the portable Agent Runtime core broad enough for other runtimes while adding the Lime AgentRuntime Profile as the strict conformance layer for Lime current runtime.

## Highlights

- Adds the Lime AgentRuntime Profile in English and Simplified Chinese.
- Adds stricter Lime profile event and snapshot schemas.
- Adds profile fixtures for submit turn, tool approval, task retry, routing single candidate, evidence export, and thread read snapshot.
- Clarifies public core, recommended extensions, and product profile conformance layers.
- Refreshes README, navigation, release notes, package metadata, and LLM entrypoints for v0.4.0.
